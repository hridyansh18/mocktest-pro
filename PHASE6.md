# Phase 6 — Create Test, Question Manager and Bulk Import UI

Implemented:
- Full Create Test form connected to POST /api/admin/tests
- Test details, scheduling, scoring, negative marking, access, shuffle and result settings
- Automatic navigation into Question Manager after test creation
- Test selector and question list
- Add/edit/delete/duplicate question UI
- Correct-answer radio selection
- Marks, negative marks, difficulty and explanation controls
- Question reorder controls connected to backend
- CSV and JSON bulk import modal connected to backend
- Bulk import source preview/edit area and error feedback
- Responsive premium admin styling

Phase 7 remains: student access/instructions/exam UI, timer, autosave, anti-cheat hooks, submit and result screens.
